# galactic-bangle

NES-style browser game show: **Who Wants to Zorp A Blorginaire?**

## Run locally

Open `index.html` directly in a browser, or serve the folder:

```bash
python3 -m http.server 8000
```

Then visit <http://localhost:8000>.

## Gameplay

- Wormhole intro sequence to the alien stage.
- 10 escalating absurd trivia questions about humanity.
- You can never lose: the alien host accepts every answer.
- 5-minute show timer and score tracking.
- Massive dance party finale.
- Replay support with multiple question sets for variety.
